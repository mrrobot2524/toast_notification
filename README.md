Toast Notification System 🔔

A simple Toast Notification UI component built using HTML, CSS, and Vanilla JavaScript.
This project demonstrates how to create dynamic notifications such as Success, Error, Warning, and Info messages.

📌 Features

Simple toast notification system

Multiple notification types:

✅ Success

❌ Error

⚠️ Warning

ℹ️ Info

Dynamic creation using JavaScript

Auto-hide notifications

Clean and minimal UI

Easy to integrate into any web project
